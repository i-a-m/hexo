Arduino和HC-SR04超声波传感器测距

Jun 26, 2017 | Arduino | 阅读

HC-SR04 立即购买

一款利用超声波测距离的传感器，多应用于 机器人避开障碍物，距离测量。

其模块，用Trig触发测距

会发出8 个 40khz的方波，自动检测是否有信号返回

有信号返回，通过echo输出高电平，高电平持续的时间就是距离的2倍

测量距离 = （高电平时间*声速）/ 2

主要技术参数

1：使用电压：DC—5V

2：静态电流：小于2mA

3：电平输出：高5V

4：电平输出：底0V

5：感应角度：不大于15度

6：探测距离：2cm-450cm

7: 高精度 可达0.2cm
