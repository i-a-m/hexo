本教程适合于有一定编程经验的同学，使用Python3，在Jupyter进行调试开发。

涉及的Python基础包括：

  • 变量和函数的定义和使用
  • 列表和字典等数据结构的使用
  • 条件和循环语句，if、for等
  • 模块的导入和使用，import语法

需要安装以下依赖库：

  • jupyter - 交互式笔记本
  • matplotlib - Python2D绘图库
  • jieba － Python中文分词组件
  • pillow - Python图像处理库
  • wordcloud － Python词云库
