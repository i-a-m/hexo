说明

  • 介绍如何实现根分区扩展到整张SD卡

步骤

  • 安装raspi-config(已安装忽略)
  • 运行raspi-config
  • 界面选择，Expand Filesystem （也有版本为expand_rootfs）， 选择并确定后
  • 查看空间
  • 效果，/dev/root之前为7G，现在增加到15G
